Non Terminal Mac Shortcuts:
Fn + Ctrl + F : Maximize but don't full screen
Fn + Ctrl + R: Return to Previous Size

Command + M: Minimize

Alt + Tab: Navigate between windows
|
->(while cycling Windows) Press, hold, and release Option to un-mimize a window

Fn + Ctrl + Arrow Key: 					Move window to Quadrant
Option + Arrow Key:  					Move to next word in direction
Option + Shift + Arrow Key: 			Select Whole Word 

Command + Arrow Key:					Move to Start/End of Line / File
Command + Shift + Arrow Key:			Select Entire Direction
